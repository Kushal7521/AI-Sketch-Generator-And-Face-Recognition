# AI Sketch Generator & Face Recognition 🕵️‍♂️🎨

## Overview
An intelligent web platform designed to bridge the gap between digital image processing and forensic-style facial reconstruction. This application allows users to generate high-quality sketches from standard images and features a facial recognition system for matching and analysis. 

Built with a modern web stack, this project demonstrates the integration of complex AI/ML processing pipelines within a fast, responsive user interface.

## Key Features
* **Intelligent Sketch Generation:** Transforms uploaded photographs into detailed, forensic-style sketches using AI models.
* **Facial Recognition & Matching:** Analyzes facial features to identify or match individuals across different image inputs.
* **Modern UI/UX:** A highly responsive, dark-mode accessible interface built with modern React patterns.
* **Environment Security:** Secure handling of API keys and AI service endpoints using local environment variables.

## Tech Stack
Based on the repository configuration, this project utilizes:
* **Framework:** [Next.js](https://nextjs.org/) (React)
* **Styling:** Tailwind CSS (configured via `postcss.config.mjs`)
* **Components:** UI components managed via `components.json` (e.g., shadcn/ui)
* **Configuration:** TypeScript ready (`tsconfig.json`) / JavaScript
* **Package Manager:** npm (`package-lock.json`)
